Blog con Django
Este es un proyecto de blog desarrollado en Django como parte del curso de Python y Django en Coder House.

Cómo utilizar el blog
Para acceder al blog, se debe crear una cuenta de usuario haciendo clic en "Registrarse" en la barra de navegación superior.

Se debe completar el formulario de registro con un nombre de usuario, contraseña y correo electrónico.

Una vez registrado, se puede acceder al blog con las credenciales creadas en el paso anterior.

Para crear una nueva entrada en el blog, se debe hacer clic en "Crear entrada" en la barra de navegación superior.

En la página de creación de entradas, se debe ingresar el título y el contenido de la entrada en el formulario.

Finalmente, se debe hacer clic en "Publicar" para guardar la entrada y mostrarla en la página de inicio del blog.

Si se desea cambiar la información del usuario, se debe hacer clic en "Perfil" en la barra de navegación superior.

En la página de perfil, se puede hacer clic en "Actualizar perfil" para cambiar el nombre de usuario, correo electrónico, contraseña y/o imagen de perfil.

Para actualizar la imagen de perfil, se debe hacer clic en "Examinar", buscar la imagen deseada y hacer clic en "Actualizar".

Requisitos del sistema
Para ejecutar el proyecto de blog, se necesita tener instalado Python 3 y Django 3 en el sistema.
